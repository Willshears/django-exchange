About


Website description:

I created this site as a portfolio and to experiment with side projects I am working on. The goal when creating this was to mostly use Django features to keep the code maintainable and scalable. The front end uses bootstrap to ensure it's responsive on any device. My first project on here is a bug tracker application, which uses a very similar set up to a blog or social media site in the way it has been configured. With the sim exchange, the prices were previously scraped from CoinDesk using BeautifulSoup 4, but now they are taken directly from the Binance API. As of 15/7/2022 the token prices are rendered on a 1 second ticker, displaying live prices in a similar way to the leading exchanges. For an extra layer of validation, two functions are used for the price, so the market price variable is calculated at the end of the buy/sell form validation process. This ensures that the system cannot be exploited.


Upcoming features:

  • Content posting for non admin accounts
  • REST API integration
  • Upgraded user profiles
  • Simulated trading model integration
  • Custom charts: currently they are provided by TradingView, which displays a different value to Binance as they compile data from a variety of exchanges.
